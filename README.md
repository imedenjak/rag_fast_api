# RAG Agent 🤖

A Retrieval-Augmented Generation (RAG) Agent built with LangChain, LangGraph, Qdrant, and Streamlit, containerized with Docker.

## Overview

This project loads content from a web page, splits it into chunks, stores embeddings in **Qdrant** vector store, and exposes a **Streamlit chat UI** to answer questions using a **LangGraph agent** with RAG + Reciprocal Rank Fusion.

The agent:
1. Retrieves relevant documents using multi-query + reciprocal rank fusion
2. Generates an answer from retrieved context
3. Grades the answer for hallucinations — retries if not grounded

## Tech Stack

- **LangChain** — RAG pipeline orchestration
- **LangGraph** — Stateful agent with grading and retry logic
- **Qdrant** — Persistent local vector store
- **OpenAI** — Embeddings (text-embedding-ada-002) + LLM (GPT-3.5-turbo)
- **Streamlit** — Chat UI
- **Docker** — Containerization

## Agent Flow

```
User Question
      ↓
[retrieve]        → multi-query + reciprocal rank fusion
      ↓
[generate]        → answer from retrieved context
      ↓
[grade_answer]    → is answer grounded? no hallucination?
      ↓
 grounded? ──YES──► END
     │
     NO
     └──────────► [generate] retry
```

## Getting Started

### 1. Clone the repository

```bash
git clone <your-repo-url>
cd my_rag
```

### 2. Set up environment variables

```bash
# Copy the example env file
cp .env.example .env

# Edit .env and add your OpenAI API key
OPENAI_API_KEY=sk-xxx
```

### 3. Install dependencies locally

```bash
# Activate virtual environment
.\venv\Scripts\Activate.ps1   # Windows
source venv/bin/activate       # Linux / Mac

# Install
pip install -r requirements.txt
```

### 4. Start Qdrant

```bash
docker compose up qdrant -d
```

### 5. Run ingestion — build Qdrant index (once only)

```bash
cd app
python ingest.py
```

Expected output:
```
Loading documents...
Loaded 1 documents
Splitting documents...
Created 80 chunks
Embedding and saving to Qdrant...
Saved 80 vectors to Qdrant
Ingestion complete!
```

### 6. Run Streamlit locally

```bash
streamlit run streamlit_app.py
```

Open browser at:
```
http://localhost:8501
```

## Or Run Everything in Docker

## When to Re-run ingest.py

```bash
cd app
python ingest.py
```

Re-run when:
- Indexing a different or updated web page
- Changing chunk size or overlap settings
- Switching embedding models (delete qdrant_db/ first)

## Reset Vector Store

```bash
# Windows PowerShell
Remove-Item -Recurse -Force .\app\qdrant_db\

# Then re-ingest
cd app
python ingest.py
```

## Notes

- `@st.cache_resource` builds the agent once — not on every user interaction.
- OpenAI embeddings use `text-embedding-ada-002` with **1536 dimensions**.
- The agent retries generation if the answer is not grounded in retrieved context.
- Re-ingestion required when switching embedding models — dimensions must match.
